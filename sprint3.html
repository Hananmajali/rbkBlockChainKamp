Getting Started and Project Set Up

npm

This sprint uses npm to manage its dependencies. npm comes bundled with Node, and is another JavaScript package manager that makes it easy for developers to share and reuse code. Even though npm started in the Node ecosystem, it is quickly becoming the default choice for sharing all types of JavaScript code. Install Node.js if you haven't yet.

Run npm install. 

Solc

JavaScript bindings for the Solidity compiler. 

Run npm install solc —save. 

npm scripts

npm comes with a handy scripting feature that lets you define shell commands inside of package.json.

The Project

  • Inside of your repo:

    First, get familiar with the EIP20Interface.sol file and try to understand its functions with their parameters. 

    * Don’t add/delete comments in this file or you will get errors because they are considered part of the code

    Run npm run test. 

    To pass the tests:  

    • Start implementing the EIP20Interface.sol functions in HiddenFig.sol file

      • * Hint: The commented line in compile.js file will help you to fix your contract errors.

    • Implement getTokens(), getEthers() functions which will let the sender buy and sell his tokens.
